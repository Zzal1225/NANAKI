# 습관 (habit)

경로: `/habits`  
**상태: 작업 중** (MVP 미완성)

---

## 목적

좋은/나쁜 습관 **일일 체크**. (예정) 점수·스트릭.

---

## 포함

- 습관 정의 (emoji · color · good/bad)
- 날짜별 체크 · 월 네비

## 예정

- 점수 · 리텐션 UI · 오늘 할 일 연동 강화

---

## 제외

| 제외 | 이유 |
|------|------|
| 소셜 랭킹 | 1인 제품 |
| 웨어러블 연동 | 범위·권한 |
| “안 했다고” 푸시 | 알림 후순위 (로컬 할 일 먼저) |
